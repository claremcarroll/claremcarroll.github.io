Paper Airplane

A Leap Motion driven flying game. The user moves the paper airplane around the universe with hand movement. The objective of this game is to collect as many coins as you can in one minute.

Tools: Processing, Maya, Illustrator

Mickey Mirror

A software mirror where you can try on Mickey Mouse Ears

I wanted to create an experience that could be implemented in a theme park setting. Disneyland has a plethora of different “ears” that you can purchase throughout the park, but not all of them are available at each location. The objective of this project was to make an interactive interface that could be installed throughout the park for guests to order any hat and pick it up on their way out.

Tools: Illustrator, Processing

AR Alphabet

An augmented reality alphabet designed to make it fun to learn how to spell. Alphabet now calles out to a dictionary API and will animate the letters if they spell out a real word.

Tools: Processing, Font Constructior, Illustrator

Genesis Diyed

A redundant augmented reality board project in which users can draw their own characters and scan them into a 3D universe. Using the augmented reality markers, users can zoom in on their characters and plant trees around the universe. This project was featured at NYU's ITP showcase in winter

Tools: Processing, Maya, Photoshop, Illustrator

Collaborators: Yuji Tsuchikawa and Rock Zou
